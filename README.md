# Kazend

**The Platform That Puts Realtors First**

Kazend is a modern CRM platform designed specifically for real estate agents. It provides a comprehensive solution to manage clients, properties, and deals all in one elegant interface.

## Features

- **Client Management** - Keep track of all your clients, their preferences, and communication history in one centralized hub
- **Property Listings** - Organize and manage your entire property portfolio with rich media galleries and detailed analytics
- **Deal Pipeline** - Track every deal from first contact to closing with an intuitive Kanban-style pipeline view

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS framework
- [TypeScript](https://www.typescriptlang.org/) - Type-safe JavaScript

## Project Structure

```
src/
├── components/
│   ├── Header.astro      # Site header with logo and launch badge
│   ├── Footer.astro      # Site footer with links
│   └── FeatureCard.astro # Reusable feature card component
├── layouts/
│   └── Layout.astro      # Base layout with meta tags and global styles
├── pages/
│   └── index.astro       # Landing page
└── styles/
    └── global.css        # Global styles and Tailwind configuration
```

## Design

The website features a refined, luxury aesthetic with:

- Dark charcoal background with subtle noise texture
- Gold accent colors for premium feel
- Syne display font paired with Outfit body font
- Geometric grid pattern overlay
- Smooth animations and hover effects
- Fully responsive design for desktop and mobile

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```
